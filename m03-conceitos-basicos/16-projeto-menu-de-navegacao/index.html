PROJETO: MENU DE NAVEGAÇÃO

OBS.: Ver anotações no arquivo scipt.js


1 - Um OBJETO é um tipo especial de variável que contém PROPRIEDADES e MÉTODOS.

2 - Um OBJETO agrega diversos valores (valores primitivos, outros objetos, funções...) e permite armazenar e recuperar esses valores pelo nome.

3 - OBJETOS são dinâmicos.

4 - Uma PROPRIEDADE tem um nome e um valor.

5 - Todo objeto tem três ATRIBUTOS DE OBJETO associados:
• O PROTÓTIPO de um objeto é uma referência para outro objeto do qual as propriedades são herdadas.
• A CLASSE de um objeto é uma string que classifica o tipo de um objeto.
• O flag EXTENSÍVEL de um objeto especifica se novas propriedades podem ser adicionadas no objeto.
