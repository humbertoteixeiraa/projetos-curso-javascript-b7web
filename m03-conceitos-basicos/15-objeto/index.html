OBJETO

1 - Um ARRAY é um conjunto ordenado de valores. Cada valor é chamado de ELEMENTO e cada elemento tem uma posição numérica no array, conhecida como ÍNDICE.

2 - Os ELEMENTOS de um ARRAY podem ser de tipos diferentes: variáveis, objetos, funções, arrays...

3 - Os ARRAYS em JavaScript são DINÂMICOS: eles crescem ou diminuem conforme o necessário e não há necessidade de declarar um tamanho fixo para o array ao criá-lo ou realocá-lo quando o tamanho muda.
